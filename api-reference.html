API Reference

Dokumentasi lengkap untuk mengintegrasikan andalanKU dengan sistem Anda

Authentication

Semua request API memerlukan token autentikasi yang valid. Token dapat diperoleh melalui endpoint authentication.

Penting: Jaga kerahasiaan API key Anda dan jangan pernah membagikannya kepada pihak lain.

Generate API Token

POST /api/v1/auth/token
curl -X POST https://api.andalanku.com/v1/auth/token \ -H "Content-Type: application/json" \ -d '{"username": "your_username", "password": "your_password"}'

Parameters

Parameter Tipe Deskripsi
username Required string Username akun andalanKU Anda
password Required string Password akun andalanKU Anda

Response

{ "status": "success", "data": { "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...", "expires_in": 3600 } }

Employees API

API untuk mengelola data karyawan dalam sistem andalanKU.

List Employees

GET /api/v1/employees
curl https://api.andalanku.com/v1/employees \ -H "Authorization: Bearer your_token"

Query Parameters

Parameter Tipe Deskripsi
page integer Nomor halaman untuk paginasi (default: 1)
limit integer Jumlah item per halaman (default: 10)

Inventory API

API untuk manajemen inventaris dan stok barang.

Get Stock Level

GET /api/v1/inventory/stock
Tips: Gunakan parameter low_stock untuk mendapatkan daftar barang dengan stok minimum.
curl https://api.andalanku.com/v1/inventory/stock \ -H "Authorization: Bearer your_token"

Financial API

API untuk manajemen transaksi keuangan dan laporan.

Create Transaction

POST /api/v1/financial/transactions
curl -X POST https://api.andalanku.com/v1/financial/transactions \ -H "Authorization: Bearer your_token" \ -H "Content-Type: application/json" \ -d '{"type": "income", "amount": 1000000, "description": "Payment received"}'

Reports API

API untuk mengakses dan menghasilkan berbagai jenis laporan.

Generate Report

POST /api/v1/reports/generate
Tips: Laporan dapat dihasilkan dalam format PDF atau Excel sesuai kebutuhan.
curl -X POST https://api.andalanku.com/v1/reports/generate \ -H "Authorization: Bearer your_token" \ -H "Content-Type: application/json" \ -d '{"type": "financial", "format": "pdf", "date_range": "monthly"}'
